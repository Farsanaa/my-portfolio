Project Overview

This mini automation project was created to explore how embedded systems and mechanical motion can work together to create a visually engaging logo reveal.

The model uses a wave gesture to trigger the opening action. When the gesture is detected, the box door opens, the logo inside is revealed, and lighting effects improve the presentation. The project was completed in a short time as a smaller version of a bigger concept involving an automatic sliding door, fog effect, and forward logo movement.

The prototype was built using an Arduino board, servo motor control, and LED lighting. It also gave practical experience in combining electronics, mechanics, and presentation design into one working model.

Key Features

Wave Gesture Detection

The system begins the logo reveal sequence when a hand wave is detected.

Servo-Controlled Door

A servo motor opens the box door smoothly to reveal the logo placed inside.

Arduino Automation

The Arduino board manages the timing, motion control, and system response.

LED Reveal Effect

Lighting effects enhance the visual impact of the logo presentation inside the box.
