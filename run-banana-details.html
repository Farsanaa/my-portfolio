Project Overview

Run NaNaNa is a simple dino-inspired web game built as a fun interactive project. The game focuses on timing, movement, and a playful visual style, giving users a light and engaging browser-based experience.

This project helped strengthen practical skills in front-end development by combining layout design, JavaScript-based interactions, and game-style mechanics inside a web application.

Key Features

Arcade-Style Gameplay

A simple runner concept inspired by classic endless browser games.

Fast Interaction

Built around quick reactions, movement, and obstacle-based challenge.

Playful Visual Design

Uses a fun and creative look to make the experience more engaging.

Front-End Practice

Improved skills in web-based animation, interface design, and interactive logic.
