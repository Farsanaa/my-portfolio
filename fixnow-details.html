Project Overview

FixNow is a web-based application designed to help users report issues and connect with nearby service support in an organized and practical way.

Built as a hackathon project, it highlights web development skills through clear workflows, a user-friendly interface, and structured issue management for faster communication and response.

Key Features

Issue Submission

Users can quickly raise service requests and report problems through the app.

Service Discovery

The concept focuses on helping users find nearby service workers efficiently.

Responsive Interface

The pages are designed to stay simple, clean, and easy to use across devices.

Structured Management

The workflow keeps issue handling organized for better tracking and coordination.
